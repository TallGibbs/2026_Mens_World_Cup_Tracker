# Daily snapshots

Dated, read-only copies of the tracker, one per daily update run, named
`world_cup_tracker_YYYY-MM-DD.html`.

Each file is a fully self-contained snapshot of the tracker as it looked on that
date (markup, styling, and JavaScript all inline), so any copy can be opened
directly in a browser on its own.

The live, always-current tracker lives at the repository root in
`world_cup_tracker.html`. This folder is only for the dated history, which keeps
the repository root from filling up as the tournament goes on.
